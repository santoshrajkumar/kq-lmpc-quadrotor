## üåü Key Features

‚úÖ **Analytical Koopman lifting** with **generalizable observables**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí No neural networks, no training, no data fitting required  

‚úÖ **Data-free Koopman-lifted LTI + LPV models**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí Derived directly from **SE(3) quadrotor dynamics** using Lie algebra structure  

‚úÖ **Real-time Linear MPC (LMPC)**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí Solved as a **single convex QP** termed **KQ-LMPC**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí < **10 ms** solve time on **Jetson NX / embedded hardware**  

‚úÖ **Trajectory tracking on SE(3)**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí **Provable controllability** in lifted Koopman space  

‚úÖ **Closed-loop robustness guarantees**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí **Input-to-state practical stability (I-ISpS)**  

‚úÖ **Hardware-ready integration**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí Works with **PX4 Offboard Mode**, **ROS2**, **MAVSDK**, **MAVROS**  

‚úÖ **Drop-in MPC module**  
&nbsp;&nbsp;&nbsp;&nbsp;‚Üí for both KQ-LMPC, NMPC with acados on Python.

> ‚ö†Ô∏è **Important Dependency Notice**
> This package relies on **acados** for fast Model Predictive Control (MPC).  
> You must configure **acados + Python interface** before running MPC examples.
> Recommended that you install acados and configure, before installing the package (for MPC)
> Koopman Lifting and LQR will work without acados

> **Quick Setup Checklist**
> - Install acados ‚úÖ
> - Enable Python interface ‚úÖ
> - Export `ACADOS_SOURCE_DIR` ‚úÖ
> - Set library paths:
>   - Linux: `LD_LIBRARY_PATH`
>   - macOS: `DYLD_LIBRARY_PATH`
>
> üìö Install acados: https://docs.acados.org/installation/index.html  
> üêç acados Python Interface: https://docs.acados.org/python_interface/index.html  
> üíª OS Support: **Linux/macOS** (*Not tested on Windows*)
